# HSI-Core - RUN करने के लिए

सबसे आसान तरीका:

```bash
cd /Users/anurag/Desktop/HSI-Core
bash start.sh
```

फिर select करो:
- **1** for Web Dashboard (सबसे अच्छा) → http://localhost:8000
- **2** for Desktop GUI (PyQt5)
- **3** for Python Shell

---

## Quick Commands

### Web Server
```bash
python3 server_enhanced.py
```
फिर खोलो: http://localhost:8000

### Desktop App
```bash
python3 gui_main.py
```

### Auto Scan (बिना UI के)
```bash
python3 main.py
```
